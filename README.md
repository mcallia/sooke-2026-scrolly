# Sooke, before the 2026 vote — a guided map tour

Scroll-driven map story: how the NDP+Green vote moved inside the District of Sooke
between 2013 and 2025, poll by poll, heading into the October 17, 2026 municipal election.

Single self-contained page: `index.html` (map SVG baked in).

Data: Elections BC (voting-area & voting-place results, boundary sets via BC Data Catalogue),
Elections Canada (official poll-by-poll results & polling-division boundaries,
Esquimalt–Saanich–Sooke, 2015–2025), GeoBC base mapping.
All sub-riding figures are election-day ballots only. Built July 2026.
